NIVEL BÁSICO

👋 Hola Mundo

El primer programa que todo programador escribe. Su objetivo es simple: mostrar un mensaje en pantalla. En C++ usamos cout para imprimir texto.

¿Qué aprenderás?

  • Cómo se estructura un programa en C++
  • Para qué sirve #include <iostream>
  • Qué hace la función main()
  • Cómo imprimir texto con cout
  • Qué significa return 0
hola_mundo.cpp C++ 
#include <iostream>   // Librería para imprimir
using namespace std;   // Usamos el espacio std

int main() {           // Función principal

    cout << "¡Hola Mundo!" << endl;
    // cout  → imprime en pantalla
    // endl  → salta a la siguiente línea

    return 0;            // El programa terminó bien
}
▶ SALIDA DEL PROGRAMA
¡Hola Mundo!
NIVEL BÁSICO

➕ Suma de Números

Aprende a pedirle al usuario que ingrese dos números y sumarlos. Usamos cin para leer datos del teclado y realizamos operaciones aritméticas básicas.

¿Qué aprenderás?

  • Cómo leer datos del usuario con cin
  • Declarar variables de tipo int
  • Realizar operaciones aritméticas (+, -, *, /)
  • Mostrar resultados en pantalla
suma.cpp C++ 
#include <iostream>
using namespace std;

int main() {
    int a, b, resultado;  // Declaramos variables

    cout << "Ingresa el primer número: ";
    cin >> a;              // cin lee lo que escribe el usuario

    cout << "Ingresa el segundo número: ";
    cin >> b;

    resultado = a + b;     // Realizamos la suma

    cout << "Resultado: " << resultado << endl;

    return 0;
}
▶ SALIDA DEL PROGRAMA
Ingresa el primer número: 5
Ingresa el segundo número: 3
Resultado: 8
NIVEL BÁSICO

📦 Variables y Operaciones

Las variables son "cajas" donde guardamos datos. En C++ debes indicar qué tipo de dato guardarás: números enteros, decimales, texto, etc.

¿Qué aprenderás?

  • Tipos de datos: int, float, double, string
  • Cómo asignar valores a variables
  • Operaciones con distintos tipos
  • Mostrar variables junto con texto
variables.cpp C++ 
#include <iostream>
#include <string>
using namespace std;

int main() {
    int    edad    = 20;         // Número entero
    double altura  = 1.75;      // Número decimal
    string nombre  = "Carlos";  // Texto
    bool   activo  = true;      // Verdadero/falso

    cout << "Nombre: "  << nombre << endl;
    cout << "Edad: "    << edad   << endl;
    cout << "Altura: "  << altura << "m" << endl;
    cout << "Activo: "  << activo << endl;

    return 0;
}
▶ SALIDA DEL PROGRAMA
Nombre: Carlos
Edad: 20
Altura: 1.75m
Activo: 1
NIVEL MEDIO

🔀 Condicionales (if / else)

Los condicionales permiten que tu programa tome decisiones. Si se cumple una condición, hace una cosa; si no, hace otra. Es como el cerebro del programa.

¿Qué aprenderás?

  • Sintaxis de if, else if y else
  • Operadores de comparación: >, <, ==, !=
  • Tomar decisiones basadas en condiciones
  • Anidar condiciones múltiples
condicional.cpp C++ 
#include <iostream>
using namespace std;

int main() {
    int calificacion;
    cout << "Ingresa tu calificación: ";
    cin >> calificacion;

    if (calificacion >= 90) {
        cout << "¡Excelente! Obtuviste A" << endl;
    } else if (calificacion >= 70) {
        cout << "Bien hecho. Obtuviste B" << endl;
    } else if (calificacion >= 60) {
        cout << "Pasaste. Obtuviste C" << endl;
    } else {
        cout << "Reprobado. Estudia más." << endl;
    }

    return 0;
}

Ingresa tu calificación: 85
Bien hecho. Obtuviste B
NIVEL MEDIO

🎛️ Switch / Casos

El switch es una alternativa al if/else cuando tienes múltiples opciones. Es más limpio y legible cuando comparas un valor contra muchas posibilidades.

¿Qué aprenderás?

  • Cuándo usar switch en lugar de if/else
  • Sintaxis de case y break
  • El caso default para valores no esperados
  • Menús interactivos para el usuario
switch.cpp C++ 
#include <iostream>
using namespace std;

int main() {
    int opcion;
    cout << "Menú:\n1. Saludar\n2. Despedirse\n3. Salir\n";
    cout << "Elige una opción: ";
    cin >> opcion;

    switch (opcion) {
        case 1:
            cout << "¡Hola! ¿Cómo estás?" << endl;
            break;          // Salimos del switch
        case 2:
            cout << "¡Hasta luego!" << endl;
            break;
        case 3:
            cout << "Saliendo..." << endl;
            break;
        default:           // Opción no válida
            cout << "Opción inválida" << endl;
    }

    return 0;
}

NIVEL AVANZADO

🔁 Ciclos (for / while)

Los ciclos repiten un bloque de código múltiples veces. El for se usa cuando sabes cuántas veces repetir; el while cuando dependes de una condición.

¿Qué aprenderás?

  • Sintaxis del ciclo for con contador
  • Sintaxis del ciclo while
  • Variable de control e incremento
  • Cuándo usar cada tipo de ciclo
ciclos.cpp C++ 
#include <iostream>
using namespace std;

int main() {
    // ── Ciclo FOR ────────────────────────
    cout << "Contando con FOR:" << endl;
    for (int i = 1; i <= 5; i++) {
        // i=1 → condición i≤5 → i++ al final
        cout << "Vuelta número: " << i << endl;
    }

    // ── Ciclo WHILE ──────────────────────
    cout << "\nContando con WHILE:" << endl;
    int j = 1;
    while (j <= 3) {     // Repite mientras j ≤ 3
        cout << "j vale: " << j << endl;
        j++;               // Incrementamos j
    }

    return 0;
}
